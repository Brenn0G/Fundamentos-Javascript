Estruturas Condicionais - IF e ELSE

Conteúdo

Essa estrutura condicional permite criar blocos de código que vão ser executados somente quando uma determinada operação lógica for verdadeira.

No geral pode ser entendido como:


"SE condição === true ENTAO faça uma coisa SENÃO faça outra coisa"

Sintaxe: 

                if(condição) {
                    //...
                } else {
                    //...
                }
            

                if (condição) {
                    //...
                } else if (condição 2) {
                    //...
                } else if (condição 3) {
                    //...
                } else {
                    //...
                }

Operador Ternário:

É a forma abreviada de se utilizar um IF e Else.

Sintaxe:

    const resultado = condicao ? resultado_verdadeiro : resultado_falso
