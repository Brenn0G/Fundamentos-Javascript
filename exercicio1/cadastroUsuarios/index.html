Conteúdo

Exercício de fixação do curso de Javascript com o objetivo de práticar e escrever um programa em javascript que permita salvar informações de um usuário e exibi-las na tela.

As informações a serem salvas são:

  • Primeiro nome;
  • Sobrenome;
  • Campo de estudo;
  • Ano de Nascimento.

Como testar?

Basta iniciar a página HTML para ver o funcionamento do código em JS.

Decidi utilizar o object "window" ao invés do "console" para fazer com que o JS abra os pop-ups do navegador e facilite a utilização do código.
